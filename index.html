    Welcome to Tweetk'l Stars

    The Tweetk'l Stars client reads a local JSON file and presents the information intuitively on a graphicly pleasing interface.

    The client adheres to Responsive Design principles by providing at 2 layouts depending on the width of the browser window. It is capable of managing a large number of favorites. Any links included in tweets can be opened in a new window or iframe.
