    Welcome to Tweetk'l Stars

    The Tweetk'l Stars client reads a local JSON file and presents the information intuitively on a graphicly pleasing interface.

    The client adheres to Responsive Design principles by providing at 2 layouts depending on the width of the browser window. It is capable of managing a large number of favorites. Any links included in tweets can be opened in a new window or iframe.

    Menu

    To open the panel containing, start, information and about links, click on the Menu button on the main navigation window

    Start

    To open the main page which contains the favourite tweets and details about users, click on the Start button

    Tweet and user Info

    To view tweets in greater details, click the gear icon to the right

    View Links

    In both modes, referenced pages can be opened by first selecting the gear button, then selecting open or by clicking on the tweet.

    If in landscape mode, selecting the tweet will reveal the referenced page in an iframe.

    Pagination

    Navigate through pages of your favourited tweets using the slider located at the bottom of the screen.
